## ✨ 特性

- **自动化脚本** - 提高开发效率的实用脚本
- **系统工具** - 日常系统管理和维护脚本
- **开发配置** - 优化的开发环境配置文件
- **跨平台** - 支持 Linux 和 macOS 系统

## 🚀 快速开始

### 前置要求

- Bash 4.0+
- Python 3.6+
- 常见 Linux/macOS 工具

### 安装使用

1. **克隆仓库**
   ```bash
   git clone https://github.com/liuwenju/codex.git
   cd codex

2. **设置执行权限**
   ```bash
   chmod +x shell/**/*.sh
   chmod +x python/**/*.py
4. **运行脚本**
   ```bash
   ./shell/utils/example-script.sh
   python3 python/tools/example-tool.py
